# smartgov-ui

Vue.js frontend for the [SmartGov low emission zones model](https://github.com/PaulBreugnot/SmartGovLezModel).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
